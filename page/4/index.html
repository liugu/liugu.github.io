Scrapy 命令行工具

Scrapy 命令行工具

标签（空格分隔）： 爬虫

可用的工具命令(tool commands)

通过运行命令来获取关于每个命令的详细内容:

scrapy <command> -h

查看所有可用的命令:

scrapy -h

常用命令

创建项目

scrapy startproject myproject

创建一个新的spider

scrapy genspider [-t template] <name> <domain>

scrapy genspider mydomain mydomain.com

爬取

scrapy crawl <spider>
scrapy crawl myspider

列出当前项目中所有可用的spider

scrapy list

使用Scrapy下载器(downloader)下载给定的URL，并将获取到的内容送到标准输出

scrapy fetch <url>

version

scrapy version [-v]

服务容器

什么是服务容器

服务容器，很好理解，就是装着各种服务实例的特殊类。可以通过「去餐馆吃饭」来进行类比：

  • 吃饭 - 使用服务，即调用该服务的地方

  • 饭 - 服务

  • 盘子 - 装饭的容器，即服务容器

  • 服务员 - 服务提供者，负责装饭、上饭

1.安装

pip install virtualenv

2.创建环境

virtualenv ENVNAME

3.使用此环境

source bin/activate
就是新建的环境里边有一个叫activate的脚本执行一下，windows和Linux差不多

4. 当前执行环境已经切换到刚创建的环境去了

接下来可以安装各种包都会在此独立的环境下

5. 版本问题

由于我的系统环境变量是python3,但是想创建一个python2的环境 ，才折腾的，以上步骤搞出来还是python3，
查看帮助信息
virtualenv -h

so

virtualenv --python 'D:\Python27\python.exe' python

然后进入创建的环境，执行

source bin/activate

python -V 查看，创建成功

退出虚拟环境

deactivate

rm -r venv 直接删除虚拟环境文件夹

发表于 2018-02-26 | 分类于 Linux

收集一些常用的linux命令

##

  1. 查看当前目录下各个文件或文件夹的大小
    du -h --max-depth=1 .
  2. 查找几天前的文件
    find . -maxdepth 1 -mtime +365
  3. 压缩文件
    tar -zcvf xxx.tar.gz ./*
  4. 解压文件
    tar -zxvf xxx.tar.gz .
  5. 查找包含指定内容的文件
    find ./*|xargs grep debug
  6. 批量杀死进程
    ps -ef|grep test|grep -v grep |awk '{print $2}'|xargs kill -s 9
  7. 文件切割
    split -l [行数] -d -a [文件后缀位数] [待切割文件] [文件前缀]
  8. 文件去重排序
    sort -n data.txt|uniq
  9. 递归修改目录用户组
    chown apache:apache www -R
  10. 测试并发量
    ab -n 100 -c 10 -l http://www.your_site.com
    执行POST请求
    ab -n 100 -c 10 -p post.txt http://xxx/test.php
    post.txt里面是参数，如data={“name”:”wgc”}，但是要url编码以后的参数`
