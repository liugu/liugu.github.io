发表于 2019-03-20 | 分类于 backend

list

list 是一种有序的集合，可以随时添加和删除其中的元素。

可以使用索引访问list中的元素
索引可以为负数 -1 表示最后一个元素

list 元素个数 len(list_name)
追加元素到末尾 list_name.append('item')
删除末尾的元素 list_name.pop()
list 元素可以嵌套，类似于二维、三维、四维……数组

tuple (元组)

一旦初始化就不能修改，使用小括号初始化

定义一个空的 tuple

定义一个只有一个元素的 tuple，需要在后边加一个 ,

tuple 的不变是指指向不变 ， tuple 可包含可变元素 比如 list
