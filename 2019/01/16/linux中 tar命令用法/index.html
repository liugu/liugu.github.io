linux tar 命令用法

常用方法

语法

tar(选项)(参数)

常用参数

参数 含义
-c –create：建立新的备份文件
-v 显示操作过程
-f -f<备份文件>或–file=<备份文件>：指定备份文件
-t -t或–list：列出备份文件的内容
-x -x或–extract或–get：从备份文件中还原文件
-j 支持bzip2解压文件
-z -z或–gzip或–ungzip：通过gzip指令处理备份文件
-C -C <目录>：在指定目录解压缩
-p -p或–same-permissions：用原来的文件权限还原文件
–exclude –exclude=<范本样式>：排除符合范本样式的文件。
