virtualenv使用

发表于 2018-03-20 | 分类于 default

1.安装

pip install virtualenv

2.创建环境

virtualenv ENVNAME

3.使用此环境

source bin/activate
就是新建的环境里边有一个叫activate的脚本执行一下，windows和Linux差不多

4. 当前执行环境已经切换到刚创建的环境去了

接下来可以安装各种包都会在此独立的环境下

5. 版本问题

由于我的系统环境变量是python3,但是想创建一个python2的环境 ，才折腾的，以上步骤搞出来还是python3，
查看帮助信息
virtualenv -h

so

virtualenv --python 'D:\Python27\python.exe' python

然后进入创建的环境，执行

source bin/activate

python -V 查看，创建成功

退出虚拟环境

deactivate

删除虚拟环境

rm -r venv 直接删除虚拟环境文件夹
